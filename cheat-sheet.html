§1 Core Resolution

Dice Mechanic

Roll d100 (percentile): one d10 = tens, one d10 = units. Range 01–100, uniform distribution.

Add modifier. Compare to Target Number.

Lower is better: roll + modifier ≤ TN = success.

TN Scale

TNDifficulty
25Trivial
35Easy
50Medium
65Hard
80Very Hard
90Near Impossible
Result TierConditionEffect
Critical SuccessRaw roll ≤ 05 (before modifiers)Extraordinary, almost lucky outcome.

Partial SuccessRoll + mod ≤ TN + 5Succeeds but with a cost or complication.
FailureRoll + mod > TN + 5Action fails.
Critical FailureRaw roll ≥ 96 (before modifiers)Catastrophic, almost unlucky outcome.
⚠️ Modifier Stacking

Multiple circumstance modifiers stack. Total modifier on any single check is capped at ±20. See Core Mechanics §7.1 for the full modifier table.

§3 Health & Stress

Stress Points (SP)

Max SP = 20 + (RES mod × 5)

RES ModMax SP
+0 (RES 1–4)20
+1 (RES 5–9)25
+2 (RES 10–14)30
+3 (RES 15–19)35
+4 (RES 20)40

Start at 0 SP. When SP reaches Max SP → Episode.

Stress Levels

LevelEffects
😌 StableNo penalty.
😐 Elevated−2 on RES checks.
😟 Stressed−5 on RES and INT checks. RES check (TN 40) each night for sleep.
😰 Overwhelmed−8 on all checks.
💥 EpisodeSP = Max SP. Breakdown. +10 TN penalty on all checks. See Mental Health.

Health Condition Track

StatePenalty
😊 HealthyNone
😐 Tired−5 on VIG checks
😫 Sick/Exhausted−10 on all checks
🤒 Injured/Ill−15 on all checks; cannot exert
🏥 CriticalCannot act without help
💀 Dead—

Advance one step at a time (max 2 steps per event, GM discretion). Recovery must proceed step by step — no skipping.

Recovery Rates

MethodEffect
🛏️ Sleep−1 SP per good night's sleep
🗣️ Social Support−2 SP per meaningful conversation
🏃 Exercise−1 SP per workout (−2 on strong success)
🛋️ Therapy−3 to −5 SP per session

See Core Mechanics §4 for health recovery (Tired → Healthy, etc.).
